### 三、功能描述
系统分为（身份）：求职者、企业、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改个人信息、修改头像、修改密码）
4. 我的简历
5. 我的职位收藏
6. 实习申请
7. 实习记录
8. 首页
9. 论坛
10. 评论留言
11. 公告查看
12. 企业一览
13. 职位查看
14. 简历投递

**企业功能：**
1. 登录
2. 注册
3. 首页
4. 个人中心
5. 论坛管理
6. 公告查看
7. 职位实习管理
8. 职位留言管理
9. 职位收藏管理
10. 实习申请管理

**管理员功能：**
1. 登录
2. 首页
3. 个人中心
4. 管理员管理
